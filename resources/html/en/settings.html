Links

This website is a SPA (Single Page Application), but you can bookmark a specific page in a specific language through URL parameters:

  • Languages
    • ?lang=en (English)
    • ?lang=ptbr (Brazilian Portuguese)
  • Pages
    • ?page=home (Home page. The default option)
    • ?page=projects (Projects)
    • ?page=blog (Blog)
    • ?page=settings (Settings)
  • Lighting mode
    • ?mode=light (Light mode. Actually this one is redundant because it is the default)
    • ?mode=dark (Dark mode)
  • You can mix these parameters by using the character &:
    • ?page=project&lang=ptbr (Project's page in Brazilian Portuguese)
    • ?lang=en&page=projects (Projects' page in English)
    • ?page=projects&lang=ptbr&mode=dark (Blog in Brazilian Portuguese, and in dark mode)
